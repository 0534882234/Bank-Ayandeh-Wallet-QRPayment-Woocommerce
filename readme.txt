درگاه پرداخت توسط کیف پول بانک آینده در سیستم فروشگاه ساز ووکامرس

== Description ==
**QRPayment** نام یک افزونه کاربردی برای سیستم فروشگاه ساز رایگان ووکامرس می باشد که شما را قادر می سازد تا براحتی امکان پرداخت سفارشات توسط کیف پول بانک آینده را برای کاربرانتان فراهم کنید.

* مشتریان شما هنگام خرید گزینه پرداخت با کیف پول بانک آینده را انتخاب می کنند و سپس پرداخت انجام می شود


== Installation ==
1. پوشه QRPayment را در مسیر /wp-content/plugins/ کپی کنید
2. افزونه را از طریق منوی افزونه ها در وردپرس فعال کنید
3. تنظیمات افزونه را در بخش پرداخت های ووکامرس انجام دهید
